# Home LLM + Smart Home Platform

A **local-first, vendor-independent home operating system** powered by:

- Home Assistant (control plane)
- A private local LLM (reasoning + education)
- On-prem storage (NAS)
- Multi-device dashboards (TV, phone, tablet)

This site documents the **architecture, build process, and operational philosophy**
behind a reliable, safe, and extensible smart home + education platform.

## Core Principles
- Local first
- Deterministic control
- Non-authoritative AI
- No cloud dependency for core functions
